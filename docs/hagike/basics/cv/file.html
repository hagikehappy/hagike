Module hagike.basics.cv.file

图像导入库与格式转换库

与图像处理相关的库主要有2个，现在阐明其区别与联系：

  1. cv2，图像格式为np.ndarray，作为向量格式，主要用于计算机视觉中对于图像的高级处理算法

  2. Image，图像格式为ImageFile或Image，作为文件格式，主要用于对图像的直接操作，如格式转换等

而图片在各层级上存在如下区别与联系：

  1. 屏幕显示：坐标原点在左上角，y轴向下，x轴向右

  2. np.ndarray：RGB图为三维，[H, W, C]，通道顺序为R, G, B；灰度图为二维，[H, W]；索引顺序为先y轴后x轴

  3. torch.tensor：卷积神经网络的期待输入为[C, H, W]，即使是灰度图也依然需要通道维度

同时，各层级上图片的数值属性也存在区别：

  1. np.ndarray：cv2读入或Image转化后默认为np.uint8类型，范围在$[0, 255]$

  2. torch.tensor：需要与待输入的网络保持格式一致，常用类型为torch.float32，范围在$[0, 1]$
